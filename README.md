# Arduino Temperature Monitoring System

## Project Overview
This project is a temperature monitoring system built using Arduino Uno R3. 
It reads temperature data from a sensor and displays real-time readings on a 16x2 LCD screen.

## Components Used
- Arduino Uno R3
- Temperature Sensor (e.g., LM35/DHT11)
- 16x2 LCD Display
- Breadboard and resistors

## How It Works
The temperature sensor outputs an analog signal to the Arduino.
The Arduino converts the signal using its ADC and processes the data.
The temperature value is then displayed on the LCD screen in real time.
